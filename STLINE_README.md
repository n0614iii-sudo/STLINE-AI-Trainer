## 🌟 主要機能

### 1. リアルタイムAI姿勢解析
- **YOLO11姿勢検出エンジン**: 毎秒30フレームの高速解析
- **危険動作検知**: 怪我リスクのある動作を瞬時に警告
- **フォーム自動採点**: 0-1.0スケールでの精密評価

### 2. パーソナライズドトレーニング
- **3段階レベル対応**: 初心者・中級者・上級者別プログラム
- **目標別カスタマイズ**: 減量・筋肥大・持久力など
- **身体制約考慮**: 膝痛・腰痛などの制約に自動対応

### 3. 包括的データ管理
- **自動セッション記録**: 回数・時間・消費カロリー
- **進捗可視化**: グラフとチャートによる成長追跡
- **改善提案AI**: データ分析に基づく具体的アドバイス

### 4. プロフェッショナルWeb管理
- **直感的ダッシュボード**: Bootstrap5ベースのモダンUI
- **ユーザー管理機能**: プロファイル・履歴・統計の一元管理
- **マルチデバイス対応**: PC・タブレット・スマートフォン

## 🚀 セットアップガイド

### 前提条件
- Python 3.9以上
- GPU推奨（CPUでも動作可能）
- Webカメラ・マイク
- インターネット接続

### 1. システム要件

```bash
OS: Windows 10/11, macOS 11+, Ubuntu 20.04+
CPU: Intel Core i5 以上
RAM: 8GB以上（推奨16GB）
GPU: NVIDIA CUDA対応（任意、推奨）
ストレージ: 5GB以上の空き容量
```

### 2. APIキー取得

#### Gemini API（必須）
1. [Google AI Studio](https://ai.google.dev/)にアクセス
2. Googleアカウントでログイン
3. 「Get API Key」をクリック
4. APIキーをコピー

#### Stream API（必須）
1. [GetStream.io](https://getstream.io/)でアカウント作成
2. 無料プラン（333,000分）を選択
3. API KeyとSecretを取得

### 3. インストール手順

```bash
# 1. プロジェクトディレクトリ作成
mkdir stline-ai-trainer
cd stline-ai-trainer

# 2. 仮想環境作成
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. 依存関係インストール
pip install -r requirements.txt

# 4. 環境変数設定
cp .env.example .env
# .envファイルを編集してAPIキーを設定

# 5. YOLOモデルダウンロード（自動）
# 初回起動時に自動ダウンロードされます
```

### 4. 起動方法

#### AIトレーナー起動
```bash
python personal_gym_trainer.py
```

#### Web管理画面起動（別ターミナル）
```bash
python gym_dashboard.py
```

ブラウザで `http://localhost:5000` にアクセス

### 5. デモデータ作成（オプション）

```bash
python demo.py create-data
```

## 📖 使用方法

### 基本ワークフロー

#### Step 1: ユーザー登録
1. Web管理画面にアクセス
2. 「新規ユーザー登録」をクリック
3. 必要情報を入力
   - 名前
   - フィットネスレベル
   - 目標
   - 身体的制約（あれば）

#### Step 2: セッション開始
1. 「セッション開始」ボタンをクリック
2. ユーザーと運動メニューを選択
3. AIトレーナーとビデオ通話開始

#### Step 3: トレーニング実施
1. Webカメラに全身が映るように配置
2. AIの指示に従ってトレーニング
3. リアルタイムでフォーム修正指導を受ける

#### Step 4: 結果確認
1. セッション終了後、自動記録
2. ダッシュボードで統計確認
3. 改善提案を確認
