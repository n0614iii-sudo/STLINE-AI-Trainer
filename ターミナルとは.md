# 💻 ターミナルとは？

## 📖 簡単な説明

**ターミナル**は、**文字でコンピューターに命令を出す画面**です。

### 例え

- **通常のアプリ**: マウスでクリックして操作
- **ターミナル**: 文字（コマンド）を入力して操作

## 🖥️ macOSでのターミナルの開き方

### 方法1: Spotlight検索（最も簡単）

1. **Command + Space** キーを同時に押す
2. 「ターミナル」または「Terminal」と入力
3. Enterキーを押す

### 方法2: Finderから

1. Finderを開く
2. 左メニューの「アプリケーション」をクリック
3. 「ユーティリティ」フォルダを開く
4. 「ターミナル」をダブルクリック

### 方法3: Launchpadから

1. F4キーを押す（またはLaunchpadを開く）
2. 「その他」フォルダを開く
3. 「ターミナル」をクリック

## 🎯 ターミナルの見た目

ターミナルを開くと、このような画面が表示されます：

```
HikarunoMacBook-Pro:~ hikarunejikane$ 
```

- `HikarunoMacBook-Pro`: コンピューター名
- `~`: 現在のフォルダ（ホームフォルダ）
- `hikarunejikane`: ユーザー名
- `$`: コマンド入力待ちの記号

## 📝 基本的な使い方

### コマンドを入力する

1. ターミナルに文字を入力
2. **Enterキー**を押す
3. コマンドが実行される

### 例: 現在のフォルダを確認

```bash
pwd
```

Enterキーを押すと、現在のフォルダの場所が表示されます。

### 例: フォルダの中身を確認

```bash
ls
```

Enterキーを押すと、現在のフォルダ内のファイル一覧が表示されます。

### 例: フォルダを移動

```bash
cd /Users/hikarunejikane/Downloads/STLINE-AI-Trainer
```

Enterキーを押すと、そのフォルダに移動します。

## 🎯 今回使用するコマンド

```bash
# 1. プロジェクトフォルダに移動
cd /Users/hikarunejikane/Downloads/STLINE-AI-Trainer

# 2. ブランチ名を設定
git branch -M main

# 3. GitHubにプッシュ
git push -u origin main
```

## 💡 コピー&ペーストの方法

### コマンドをコピーする

1. このドキュメントからコマンドを選択（マウスでドラッグ）
2. **Command + C** でコピー

### ターミナルに貼り付ける

1. ターミナルをクリック
2. **Command + V** で貼り付け
3. **Enterキー**を押して実行

## ⚠️ 注意点

- コマンドは**正確に入力**する必要があります
- 大文字・小文字は区別されます
- スペースも重要です
- コピー&ペーストを推奨します（入力ミスを防げます）

## 🎓 よく使うコマンド一覧

| コマンド | 説明 | 例 |
|---------|------|-----|
| `pwd` | 現在のフォルダを表示 | `pwd` |
| `ls` | フォルダ内のファイル一覧 | `ls` |
| `cd` | フォルダを移動 | `cd Downloads` |
| `cd ..` | 一つ上のフォルダに移動 | `cd ..` |
| `cd ~` | ホームフォルダに移動 | `cd ~` |
| `mkdir` | フォルダを作成 | `mkdir test` |
| `rm` | ファイルを削除 | `rm file.txt` |
| `clear` | 画面をクリア | `clear` |

## 📚 まとめ

- **ターミナル**: 文字でコンピューターに命令を出す画面
- **開き方**: Command + Space → 「ターミナル」
- **使い方**: コマンドを入力 → Enterキー
- **コピー&ペースト**: Command + C / Command + V

ターミナルは最初は難しく感じますが、慣れるととても便利です！
