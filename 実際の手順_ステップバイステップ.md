## 📝 手順（コピー&ペーストでOK）

### ステップ1: ターミナルを開く

1. **Command + Space** キーを同時に押す
2. 「**ターミナル**」と入力
3. **Enterキー**を押す

→ 黒い画面（ターミナル）が開きます

### ステップ2: プロジェクトフォルダに移動

ターミナルに以下を**コピー&ペースト**して、**Enterキー**を押す：

```bash
cd /Users/hikarunejikane/Downloads/STLINE-AI-Trainer
```

**コピー&ペーストの方法:**
- 上記のコマンドをマウスで選択
- **Command + C** でコピー
- ターミナルをクリック
- **Command + V** で貼り付け
- **Enterキー**を押す

### ステップ3: ブランチ名を設定

ターミナルに以下を**コピー&ペースト**して、**Enterキー**を押す：

```bash
git branch -M main
```

### ステップ4: GitHubにプッシュ（ここで認証が必要）

ターミナルに以下を**コピー&ペースト**して、**Enterキー**を押す：

```bash
git push -u origin main
```

### ステップ5: 認証情報を入力

コマンドを実行すると、以下のように表示されます：

```
Username for 'https://github.com': 
```

#### 5-1. ユーザー名を入力

- `n0614iii-sudo` と入力
- **Enterキー**を押す

次に、以下のように表示されます：

```
Password for 'https://n0614iii-sudo@github.com': 
```

#### 5-2. Personal Access Tokenを入力

⚠️ **重要**: ここでは**パスワードではなく、Personal Access Token**を入力します

- 先ほど取得した**Personal Access Token**をコピー
- ターミナルに**Command + V**で貼り付け
- **Enterキー**を押す

**注意**: ターミナルでは、パスワードやトークンを入力しても画面に表示されません（セキュリティのため）。これは正常です。入力したらEnterキーを押してください。

## 🔄 もしエラーが出た場合

### エラー1: "Username for 'https://github.com':"

→ これは正常です。ユーザー名を入力してください。

### エラー2: "Authentication failed"

→ Personal Access Tokenが間違っている可能性があります。もう一度確認してください。

### エラー3: "Permission denied"

→ Personal Access Tokenのスコープ（権限）を確認してください。`repo`にチェックが必要です。
