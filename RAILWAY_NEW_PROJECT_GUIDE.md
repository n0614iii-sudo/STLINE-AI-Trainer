# 🚀 Railwayで新しいプロジェクトを作成する方法

## 📍 プロジェクト作成の場所

Railwayで新しいプロジェクトを作成するには、以下の手順で進めます：

### 方法1: ダッシュボードから作成（推奨）

1. **Railwayダッシュボードにアクセス**
   - [https://railway.app](https://railway.app) にアクセス
   - ログインします

2. **「New Project」ボタンをクリック**
   - ダッシュボードの左上または中央に「**New Project**」ボタンがあります
   - または、サイドバーの「**+ New**」ボタンをクリック

3. **プロジェクト作成方法を選択**
   - **「Deploy from GitHub repo」**: GitHubリポジトリからデプロイ
   - **「Empty Project」**: 空のプロジェクトを作成
   - **「Deploy from Template」**: テンプレートから作成

### 方法2: サイドバーから作成

1. **左側のサイドバーを確認**
   - サイドバーの上部に「**+ New**」ボタンがあります
   - クリックすると、プロジェクト作成オプションが表示されます

### 方法3: プロジェクト一覧から作成

1. **プロジェクト一覧ページに移動**
   - 現在のプロジェクト名の横にある「**▼**」をクリック
   - 「**All Projects**」を選択

2. **「New Project」ボタンをクリック**
   - プロジェクト一覧ページの右上に「**New Project**」ボタンがあります

## 🎯 プロジェクト作成の詳細手順

### GitHubリポジトリからデプロイする場合

1. **「Deploy from GitHub repo」を選択**
2. **GitHubアカウントを認証**（初回のみ）
3. **リポジトリを選択**
   - 一覧から作成したいプロジェクトのリポジトリを選択
4. **プロジェクト名を設定**（任意）
5. **「Deploy」をクリック**

### 空のプロジェクトを作成する場合

1. **「Empty Project」を選択**
2. **プロジェクト名を入力**
3. **「Create Project」をクリック**
4. **サービスを追加**
   - 「**+ New**」→「**GitHub Repo**」または「**Empty Service**」を選択

## 📝 現在のプロジェクトとの違い

新しいプロジェクトを作成すると：

- ✅ **完全に独立した環境**になります
- ✅ **別のURL**が割り当てられます
- ✅ **環境変数**は別々に管理されます
- ✅ **デプロイ履歴**も独立しています

## 🔗 現在のプロジェクトに戻る方法

1. **サイドバーのプロジェクト一覧から選択**
   - 左側のサイドバーでプロジェクト名をクリック
   - または、プロジェクト名の横の「**▼**」から選択

2. **「All Projects」から選択**
   - プロジェクト一覧から目的のプロジェクトをクリック

## 💡 ヒント

- **複数のプロジェクト**を管理できます
- 各プロジェクトは**独立した環境**です
- **無料プラン**でも複数のプロジェクトを作成できます
- プロジェクト名は後から変更可能です

## 🎉 まとめ

新しいプロジェクトを作成するには：

1. Railwayダッシュボードにアクセス
2. 「**New Project**」ボタンをクリック（左上または中央）
3. 作成方法を選択（GitHub、空のプロジェクト、テンプレート）
4. 設定を完了して作成

これで新しいプロジェクトの準備が整います！
