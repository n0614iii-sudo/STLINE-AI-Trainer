# 🖥️ ターミナルの開き方 - 詳しい手順

## 📍 どこでやるの？

**Macの画面のどこでもOKです！**

デスクトップでも、ブラウザでも、どこでも大丈夫です。

## 🎯 ステップバイステップ

### ステップ1: Spotlight検索を開く

1. **Command + Space** キーを同時に押す
   - Commandキー（⌘）は、スペースキーの左側にあります
   - Spaceキーは、キーボードの一番下の長いキーです

2. **Spotlight検索の画面が表示されます**
   - 画面の中央に、検索ボックスが表示されます
   - この検索ボックスに文字を入力できます

### ステップ2: 「ターミナル」と入力

1. Spotlight検索の画面が開いたら、**そのまま**「**ターミナル**」と入力します
   - または「**Terminal**」と入力してもOKです
   - 入力すると、下に候補が表示されます

2. 候補の中に「**ターミナル**」または「**Terminal**」が表示されます
   - 黒いアイコンのアプリです

### ステップ3: Enterキーを押す

1. 「ターミナル」が選択されている状態で、**Enterキー**を押します
   - または、マウスで「ターミナル」をクリックしてもOKです

2. **ターミナルが開きます！**
   - 黒い画面（または白い画面）が表示されます
   - これがターミナルです

## 💡 別の方法（もしSpotlightが開かない場合）

### 方法1: Finderから

1. Finderを開く（Dockの一番左のアイコン）
2. 左メニューの「アプリケーション」をクリック
3. 「ユーティリティ」フォルダを開く
4. 「ターミナル」をダブルクリック

### 方法2: Launchpadから

1. F4キーを押す（または、Launchpadを開く）
2. 「その他」フォルダを開く
3. 「ターミナル」をクリック

## ✅ 確認方法

ターミナルが正しく開いているか確認：

1. 画面の一番下に、以下のような文字が表示されているはずです：
   ```
   HikarunoMacBook-Pro:~ hikarunejikane$ 
   ```

2. この `$` の後ろに、コマンドを入力できます

## 🎯 まとめ

1. **どこでもOK**: Macの画面のどこでも大丈夫
2. **Command + Space**: Spotlight検索を開く
3. **「ターミナル」と入力**: Spotlight検索のボックスに入力
4. **Enterキー**: ターミナルが開く

これでターミナルが開きます！
